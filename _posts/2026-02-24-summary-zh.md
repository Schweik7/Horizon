## [Linux 7.0 正式结束 Rust 实验阶段，确立其内核长期地位](https://t.me/zaihuapd/39806) ⭐️ 9.0/10

Linux 7.0 合并了由 Miguel Ojeda 提交的 Rust 相关补丁，正式宣布 Rust 实验阶段结束，确立其作为内核长期组成部分的地位。除文档更新外，该补丁还引入了 `__rust_helper` 注解，旨在优化内核在开启链接时优化（LTO）时的构建表现。 这一决定标志着系统编程领域的一次重大范式转变，向业界释放了明确的长期承诺信号，将鼓励企业加大对 Rust 内核开发的资源投入。此举将加速 Rust 在整个生态系统中的采用，通过其内存安全特性潜在地提升安全性和性能，影响数十亿台 Android 设备及众多 Linux 发行版。 `__rust_helper` 注解被专门添加到 C 辅助函数中，以便这些函数可以内联到 Rust 代码中，这对于在启用 Rust 和 LTO 的情况下构建内核的性能至关重要。这一变化是在 2025 年 Linux 内核维护者峰会上 Rust 实验被宣告结束后进行的，该实验评估了 Rust 在内核中使用的技术、流程和社会层面的适用性。
